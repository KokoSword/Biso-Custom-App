# 🦷 مندوب الأسنان — الكرخ | Baghdad Dental Rep App

تطبيق ويب (PWA) كامل لمندوب مبيعات مواد طب الأسنان في بغداد / جانب الكرخ:
دليل **209 عيادة ومركز** في 13 منطقة، مقسّمة على **6 جولات أسبوعية** (السبت → الخميس، الجمعة عطلة)، مع إدارة كاملة للمبيعات والذمم والمتابعات.

> يعمل بالكامل من ملف واحد — بدون سيرفر وبدون قاعدة بيانات. بياناتك تُحفظ على جهازك فقط.

---

## ✨ المميزات

| التبويب | ماذا يقدّم |
|---|---|
| 🧭 **الجولة** | مسار اليوم بترتيب الزيارة الأمثل، نصائح لكل منطقة (أين تركن وتمشي)، علامة ☀ للعيادات الصباحية، وتأشير الزيارات ✔ |
| 🗺️ **الخريطة** | خريطة حيّة بدبابيس مرقّمة ملوّنة حسب اليوم + خط المسار، والدبوس يتحول أخضر عند إتمام الزيارة |
| 🔍 **الكل** | بحث في كل العيادات + فلاتر: صباحي، بدون رقم، لم تُزَر، عليها ذمة، عملاء A، حكومي، ★4.5+ |
| 🛒 **المبيعات** | قائمة موادّك وأسعارك (قابلة للتعديل)، إنشاء طلبيات بالسعر والكمية والخصم، وإرسال **فاتورة واتساب** جاهزة للعيادة |
| 💰 **الذمم** | سجل الديون لكل عيادة، تسجيل الدفعات، تذكير تحصيل مؤدّب بالواتساب، وجدولة المتابعات مع تنبيه للمستحَق |
| 📊 **تقدّمي** | نسبة التغطية، مبيعات الشهر، أفضل الزبائن، تقرير CSV يفتح في Excel، ونسخة احتياطية (تصدير/استيراد) |

ولكل عيادة: 📞 اتصال مباشر · 💬 واتساب · 🗺️ توجيه لموقعها بالضبط في كوكل مابس · تصنيف A/B/C · حقل لتسجيل الرقم الناقص · جهة الاتصال والملاحظات · قوالب رسائل جاهزة باللهجة العراقية.

## 🚀 النشر على GitHub Pages (٣ دقائق)

1. أنشئ مستودعاً جديداً (Repository) — مثلاً باسم `dental-rep-app`
2. ارفع كل ملفات هذا المجلد إلى المستودع (index.html, sw.js, manifest.json, الأيقونات، README)
3. من **Settings ← Pages**: اختر **Deploy from a branch** ← الفرع `main` ← المجلد `/ (root)` ← Save
4. بعد دقيقة يصبح التطبيق متاحاً على:
   `https://اسم-حسابك.github.io/dental-rep-app/`

### 📱 التثبيت على الهاتف

- **أندرويد (كروم):** افتح الرابط ← ستظهر رسالة «تثبيت التطبيق» أو من القائمة ⋮ ← *إضافة إلى الشاشة الرئيسية*
- **آيفون (سفاري):** افتح الرابط ← زر المشاركة ← *Add to Home Screen*

بعد أول فتح، يعمل التطبيق **بدون إنترنت** (عدا صور الخريطة تحتاج نت لتحميل مناطق جديدة — والمناطق التي فتحتها سابقاً تُحفظ).

## 🔒 خصوصية البيانات

- كل بياناتك (الزيارات، الطلبيات، الذمم، الأرقام، الملاحظات) تُخزَّن في متصفح جهازك فقط (`localStorage`) — لا تُرسل لأي سيرفر.
- لذلك: **صدّر نسخة احتياطية أسبوعياً** من تبويب 📊 تقدّمي، ولا تمسح بيانات المتصفح.
- لنقل بياناتك لجهاز آخر: تصدير من الجهاز الأول ← استيراد في الثاني.

## 🔧 تحديث بيانات العيادات

بيانات العيادات (الأسماء، المواقع، الأرقام) مضمّنة داخل `index.html` في المتغيّر `DATA`.
لإضافة عيادة أو تعديل رقم بشكل دائم، عدّل هذا الـ JSON مباشرة ثم ارفع الملف من جديد.
(الأرقام التي تسجّلها داخل التطبيق تُحفظ محلياً وتَغلِب على الرقم الأصلي.)

## English Summary

A single-file installable PWA for a dental-materials sales rep covering Baghdad's Karkh side:
**209 clinics** across 13 districts organized into a **6-day weekly route plan** (Fri off), with a live Leaflet/OSM map, per-clinic call & WhatsApp actions, product/price catalog, order taking with WhatsApp invoices, debt ledger with payment tracking, follow-up reminders, A/B/C lead rating, CSV reporting, and JSON backup/restore. All data stays on-device (localStorage). Deploy via GitHub Pages: Settings → Pages → Deploy from branch → `main` / root.

**Stack:** vanilla HTML/CSS/JS · Leaflet 1.9.4 (cdnjs) · OpenStreetMap tiles · IBM Plex Sans Arabic · Service Worker offline cache.
